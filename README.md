A JUCE C++ stereo delay plugin built for Music System Programming.\
\
## Features\
\
- Circular buffer delay line\
- Delay time in milliseconds\
- Feedback\
- Dry/wet mix\
- Output gain\
- Stereo width\
- Simple JUCE slider interface\
\
## Built With\
\
- JUCE\
- Xcode\
- C++\
- VST3\
- Tested in FL Studio on macOS\
\
## Notes\
\
This project is inspired by modern studio delay plugins such as Waves Manny Marroquin Delay, but is implemented as a learning project from first principles.
